各作业详情摘要
--------------

小作业1 (Ch2): Python基础数据类型、字符串、列表、字典、控制结构、lambda

小作业2 (Ch3): NumPy数组创建、切片、运算、广播、布尔索引（共14小题）

小作业3 (Ch4): DataFrame创建、选择过滤、聚合groupby、merge/concat、时间序列重采样

小作业4 (Ch6): 沪深300数据导入→可视化→统计→对数回报率→重采样→滚动均线→交易策略
              （需自行获取沪深300数据，自行设计滚动技术指标和策略）

小作业5 (Ch8): Titanic数据预处理+sklearn分类模型，训练集准确率>80%，测试集>65%

小作业6 (Ch9): 房价预测回归问题，目标MSE越小越好（附house pricing数据集）

小作业7 (Ch10): sklearn实现K-Means聚类和PCA降维（使用iris数据集）

大作业1: 预测2030家上市公司最新季度的9个财务指标（R2评分），需要数据探索+ML训练+交叉验证

大作业2: 可转债多因子量化策略设计，要求像研报风格，含因子分析（带经济逻辑）和策略评估
         评分维度：收益、稳定性、可复现性
