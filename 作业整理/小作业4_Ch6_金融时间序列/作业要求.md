小作业4 - 第6章 金融时间序列
=============================

作业内容：

1. 数据导入与初步处理
   - 从CSV文件导入沪深300数据，并将日期列设置为DataFrame的索引。
     （本课件没有讲如何设置，自己去网上查询）
   - 对导入的数据进行初步观察，使用 .info() 和 .describe() 方法查看数据集的基本信息和统计概要。

2. 对沪深300进行可视化，包括时间序列图和收益柱状图。

3. 汇总统计
   - 计算并打印数据集的均值、标准差、最小值和最大值。
   - 使用 .aggregate() 方法自定义统计量，如范围（最大值-最小值）。

4. 时间序列变化分析
   - 计算金融资产价格的日日变化（使用 .diff() 方法）。
   - 计算变化率（使用 .pct_change() 方法）和对数回报率。

5. 数据重新采样
   - 对数据进行向下采样，将日数据重新采样为周数据和月数据。

6. 滚动统计
   - 计算金融资产价格的30日移动平均值和标准偏差。
   - 对于选定的资产，绘制原始价格和滚动统计量（如移动平均）的对比图。

7. 简单交易策略模拟
   - 根据滚动平均线设计一个简单的交易策略（如短期平均线穿越长期平均线），
     并且比较 long only 和该技术指标下的年化收益。
   - 自己设计一个函数，并以此开发一个滚动技术指标（本课件没有讲，自己去网上查询），
     然后开发一个简单的交易策略。
   - 计算并可视化根据该策略模拟得到的交易信号，并且比较 long only 和该技术指标下的年化收益。

注意：沪深300数据需要自行获取（可用tushare、akshare等）。
